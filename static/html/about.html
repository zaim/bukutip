About BukuTip

Welcome! BukuTip is a free web app that searches for and compare book prices from various bookstores and websites in Malaysia.

Which bookstores?

For now, BukuTip searches these bookstores and websties:

  • BookXcess
  • Kinokuniya Malaysia
  • MPH
  • Mudah.my
  • Times Bookstores

How's it done?

BukuTip simply does a direct search on each of the above bookstore's website for a book based on its ISBN code and parses (or scrapes) for the price value.

Note for bookstore webmasters: please contact BukuTip if you'd like to know more about how we scrape you or if you'd like your bookstore to be featured also.
